# Sistem Pendukung Keputusan Pemilihan Tempat Kost

## Deskripsi
Sistem Pendukung Keputusan (SPK) untuk membantu mahasiswa memilih tempat kost terbaik menggunakan metode AHP (Analytical Hierarchy Process) dan TOPSIS (Technique for Order Preference by Similarity to Ideal Solution) Berbasis Web.

## Metode
- AHP (Analytical Hierarchy Process)
- TOPSIS (Technique for Order Preference by Similarity to Ideal Solution)

## Tools
- PHP
- MySQL
- XAMPP
- Visual Studio Code
